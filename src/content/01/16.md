truncate(text-overflow: ellipsis)는

`글자가 부모 요소의 길이를 넘어가면 …으로 처리되는 것을 말한…`

요즘은 `mask-image`를 활용한 자연스러운 효과가 트렌드인 것 같다.

![](16-1.webp)

자연스럽게 서서히 없어지는 효과를 보여준다. 구현은 엄청 쉽다.

```css
.fade-mask {
	mask-image: linear-gradient(
		to right,
		black calc(100% - 24px),
		transparent
	);
}
```

`.fade-mask`를 wrapper에 적용하면 된다. `calc(100%-24px)`을 한 이유는 오른쪽 24px에만 적용하고 싶어서 이렇게 했다. 그냥 전체에 mask 씌우려면 black만 넣어주면 됨.

물론 text에 적용하려면 `overflow: hidden`이나 `white-space: nowrap;` 필요함.
